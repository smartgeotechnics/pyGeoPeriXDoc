# GeoPeriX Documentation

[![Documentation Status](https://readthedocs.org/projects/geoperix/badge/?version=latest)](https://geoperix.readthedocs.io/en/latest/)

This repository is the **public documentation mirror** for GeoPeriX — a
high-performance peridynamics engine for geomechanics research and engineering.

Documentation is automatically synced from the private GeoPeriX repository on
every commit to `main`.

## Read the Docs

**https://geoperix.readthedocs.io**

## Contents

| Path | Description |
|---|---|
| `docs/` | Sphinx source files (MyST Markdown) |
| `.readthedocs.yaml` | ReadTheDocs build configuration |
| `VERSION.txt` | Current version string |
